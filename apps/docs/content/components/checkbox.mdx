## Component API

| Prop           | Default | Description                                             |
| -------------- | ------- | ------------------------------------------------------- |
| isSelected     | false   | whether or not the checkbox has been selected           |
| isDisabled     | false   | whether or not the checkbox is disabled                 |
| isIntermediate | false   | whether or not the checkbox is in an intermediate state |

See React Aria [Checkbox](https://react-spectrum.adobe.com/react-spectrum/Checkbox.html#checkbox) for more details.
