## Component API

| Prop           | Default | Description                                             |
| -------------- | ------- | ------------------------------------------------------- |
| disabled     | false   | whether or not the checkbox has been selected           |
| htmlFor     | string   | Associated the label with a form control                 |
| required | false   | whether or not to indicate if the label is required |
| description | string   | Additional description text associated to the label |
