## Component API

| Prop       | Default    | Description                                                           |
| ---------- | ---------- | --------------------------------------------------------------------- |
| horizontal | 'vertical' | determines the layout `flex-row` or `flex-col` of the checkbox items. |
| isDisabled | false      | Can be applied on an item by item basis or on the group level.        |
| isInvalid  | false      | Marks the group as invalid with `data-invalid`.                       |

See React Aria [CheckboxGroup](https://react-spectrum.adobe.com/react-spectrum/CheckboxGroup.html) for more details.
